# Teacher Package Overview

## Package goal
Provide one strong first PQC challenge that is:
- practical,
- browser-based for students,
- realistic enough to teach protocol migration,
- and structured enough for classroom use.

## Core challenge
**Operation HARVEST BREAKER — The Quantum Gate**

Students investigate a TLS service from a browser shell and recover a mission token that is hidden in certificate metadata, not in the page body.

## What students receive
- `student/01_Briefing_Operation_HARVEST_BREAKER.md`
- `student/02_TLS_and_PQC_Primer.md`
- `student/03_Hints.md`
- a generated access card from `deploy/make_access_card.sh`

## What teachers keep private
- teacher rationale
- solution guide
- deployment guide
- infrastructure guide
- troubleshooting guide
- AI-resistance notes
- grading rubric
