# Browser Shell Orientation

When you log into the shell, try these commands first:

```bash
lab-info
help-lab
protocol-reminder
```

These commands are meant to orient you, not solve the challenge for you.

Useful environment variables:
- `LAB_HOST`
- `LAB_PORT`

Useful tools:
- `openssl`
- `grep`
- `sed`
- `less`
