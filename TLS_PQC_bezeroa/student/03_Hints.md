# Hints — Use Only If Needed

## Hint 1
Do not focus only on fetching a page body.
Something important happens before protected application data begins.

## Hint 2
The certificate matters, but not in the same way that key agreement matters.

## Hint 3
You may need to inspect the connection and then inspect certificate metadata.

## Hint 4
The reward is not hidden in the obvious application response.

## Hint 5
If you are using OpenSSL, think about the subcommand that inspects a TLS connection.
